## Setup

### Prerequisites
- Python 3.8+
- pip3
- (Optional) virtualenv

### Installation
1. Clone the repository:
```bash
git clone https://github.com/absolutecoder01/plugin-hub-minecraft.git
cd plugin-hub-minecraft
```
2. (Recommended) Create a virtual environment:
```bash
python3 -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
```
3. Install dependencies:
```bash
pip3 install -r requirements.txt
```
4. Run the app:
```bash
python3 app.py
```
5. Open in your browser:
```bash
http://127.0.0.1:5000/
```
## Usage

1. Click **"Add Plugin"** to upload a new plugin (`.jar` or `.zip`)
2. Fill in the details: name, description
3. Upload a thumbnail (PNG/JPG) for visual preview
4. Browse your collection on the Minecraft-style grid
5. Edit or delete plugins anytime

> Tip: Use consistent naming for easier organization!

## Key Features:
- Plugin upload (JAR, ZIP)
- Adding thumbnails, files
- Minecraft-style interface
- Local hosting – full control over your data
